## Running the project

 ```shell
- Ruby - 3.1.2
 - Rails - 7.0.3.1
```
## :information_source: Step by Step
1. Create the database
   ```
   rails db:create
   ```
2. Run the migrations
   ```
   rails db:migrate
   ```
3. Install gems and dependencies
   ```
   bundle install
   ```
   ```
   yarn install
   ```
4. Run the tests
   ```
   rspec
   ```
5. Run the application
   ```
   rails server
   ```
